# vocab_training

一个纯 HTML + CSS + JavaScript 实现的日语单词读音测验小工具。

## 功能

- 批量输入词表，格式：`汉字,平假名读音`
- 支持日语 / 英语切换
- 看字输入读音
- 听力听写
- 听懂判断
- 答错词进入下一轮继续测验
- 复习当前未通过单词
- 近期错误收藏夹
- 本地词汇库，按错误次数排序

## 运行方式

直接用浏览器打开：

```text
vanilla-quiz/index.html
```

不需要后端，不需要安装依赖。
